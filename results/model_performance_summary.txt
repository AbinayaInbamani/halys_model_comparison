PERFORMANCE METRICS EXPLANATION
================================

Accuracy:  (112 + 13) / 183 = 0.683
  → Misleading! Dominated by majority class (non-flight days)
  → Don't use accuracy for imbalanced data

Precision: 13 / (13 + 51) = 0.203
  → When model predicts "flight", it's correct 20% of the time
  → 80% are false alarms (acceptable for early-warning)

Recall:    13 / (13 + 7) = 0.650
  → Model detects 65% of true flight events
  → Misses 35% (7 out of 20)
  → For early-warning, want recall ~90%+

F1-Score:  2 × (0.203 × 0.650) / (0.203 + 0.650) = 0.310
  → Balanced metric when precision/recall matter
  → Indicates precision-recall tradeoff

ROC-AUC:   0.739
  → Probability that model ranks random flight event higher than
    random non-flight event
  → Good discrimination ability (0.5 = random, 1.0 = perfect)
